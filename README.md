## Usage

1. Open Git Bash Terminal
Default port & ip
```sh
sh app.sh
```
Custom port & ip
```sh
sh app.sh --port (port here) --ip (iphere) - without ()
```
there is some restrictions about the ip it must be : localhost or 127.0.0.1 or your device ip
